## 🚀 Overview
Ventoy for Windows lets you boot ISO files directly from a USB drive instead of burning each one separately. You prepare the drive with Ventoy a single time, then treat it as ordinary storage: copy any ISO, WIM, IMG, VHD or EFI file onto the stick and it is immediately ready to boot. Power on from the USB and Ventoy shows a menu of every image, so you can launch a Linux live session, a Windows installer or a rescue tool in seconds. It supports both Legacy BIOS and UEFI, adds Secure Boot support for modern systems, and is developed openly as a free project on GitHub. The result is a fast, flexible way to keep all your bootable images in one place and add new ones with a simple copy.

## ✨ Features
- **Direct ISO booting** — copy an image onto the Ventoy USB and boot it as-is; there is no burning step and no need to unpack anything.
- **Many images, one stick** — store as many ISO, WIM, IMG, VHD and EFI files as the drive holds and pick one from the boot menu each time.
- **Secure Boot support** — Ventoy Secure Boot lets your images start on modern UEFI PCs while keeping firmware protection enabled.
- **BIOS and UEFI compatible** — a single Ventoy drive boots on both older and newer machines without extra setup.
- **Large-file support** — exFAT and NTFS formatting accommodates hefty Windows installers and images, an important detail when users weigh Ventoy vs Rufus.

## 🎯 Getting Started
1. Insert your USB drive and open Ventoy on your Windows PC.
2. Select the drive and click **Install** — this erases the USB, so back up important files first.
3. When installation completes, open the Ventoy drive in File Explorer.
4. Copy your ISO, WIM, IMG or VHD files onto the drive.
5. Restart, boot from the USB, and pick an image from the Ventoy menu to run it.

## ❓ FAQ
**Is Ventoy free?**
Yes. Ventoy is free and open source, and its complete source code is available on GitHub for anyone to inspect or improve.

**Which versions of Windows are supported?**
Ventoy runs on 64-bit Windows editions from Windows 7 through Windows 11. A currently supported release is recommended.

**Is Ventoy safe to use?**
Being open source, Ventoy can be reviewed by anyone, and its Secure Boot support enables booting on protected UEFI hardware. Keep in mind that installing Ventoy wipes the target USB, so back it up beforehand.

**Ventoy vs Rufus — which should I use?**
Rufus is ideal for writing a single image to a drive, while Ventoy is built to hold and boot many ISOs from one stick. If you regularly work with multiple images, Ventoy tends to save the most time.

**Is my data secure?**
Ventoy works locally and requires no account to create your boot drive. The install step clears the USB, so back up its contents first; anything you copy on later simply lives on the drive as normal files.
